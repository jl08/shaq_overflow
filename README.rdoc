<p>A clone of the popular website Stack Overflow.</p>

<p>User can create,edit, and delete questions, answers and responses.</p>
<p>User can create answers to questions.</p>
<p>User can create response to questions.</p>
<p>User can create response to answers.</p>
<p>Users can upvote questions and answers.</p>

<p>Created using Ruby on Rails within 3 days. </p>
<p>Created using Test-Driven and Behavior Driven Development. </p>
